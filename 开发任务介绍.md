# 1.开发环境：

使用当前项目下的venv虚拟环境，相关的python库都安装在当前项目的venv虚拟环境下

# 2.任务介绍

        编写一个的书籍信息查询网站

# 3.功能实现介绍

## 3.1 后端

        后端由Django+Gunicorn+sqlite编写，sqlite数据库创建书籍信息表，该表包括字段：ISBN、书名、作者、出版时间、出版社、简介、目录共七个字段，ISBN设为主键

## 3.2 前端

        前端在浏览器中打开，上面有一个搜索框，可以切换搜索方式为ISBN搜索/书名搜索/作者搜索，三种搜索方式，搜索书名的话就去书名字段搜索，搜索作者的话就去作者字段搜索，搜索书名或作者名的时候如果存在字母，则忽略字母大小写，同时点击全部时间按键会弹出时间段选择界面，在该界面可以选择起始时间和结束时间，从而选择一个时间端，比如2016-2019或者2019-至今，我们可以通过该功能筛选出时间段内的书籍。

        查询结果没有就不显示，有的话以列表的形式显示。点进搜索结果的书籍信息界面从上到下依次显示：书名、ISBN、出版时间、出版社、简介、目录等相关信息
